[예제3.] 자동 HTML 폼 검증

[HTML 입력 검증 속성]
*disabled: 입력 요소 비활성화
*max: 입력 요소의 최댓값
*min: 입력 요소의 최솟값
*pattern: 입력 요소의 값 패턴=> 특정 정규식과 일치하도록 제한
*required: 필수 입력 필드
*type: 입력 요소의 유형

[required 예시]
* 빈칸으로 제출하기 클릭하면, 브라우저가 자동으로 오류 메세지 표시

[pattern 예시] 입력값 패턴이 맞지 않을 경우 기본 오류 메세지 띄우기
* pattern="\d{3}" : 숫자 3자리만 입력 가능하도록 제한
* pattern="\d{3}" : title="" => 입력값 패턴이 맞지 않을 경우 나타나는 기본 오류 메시지
