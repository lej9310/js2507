새로운 HTML 요소 만들기

노드 생성 - .appendChild()

* HTML DOM에 새로운 요소 추가하려면, 먼저 노드 만든 다음 => 기존 요소에 추가
* appendChild(): 새 요소를 부모의 마지막 자식으로 추가

const para = document.createElement("p") => p 태그 생성
const node = document.createTextNode("[NEW] 문단 3.") => 텍스트 노드 생성
para.appendChild(node) => 텍스트 노드를 p 태그에 추가
const elmt = document.getElementById("p2") => 기존 요소 찾기
elmt.appendChild(para) => 기존 요소에 새 요소 추가

원하는 위치에 새 요소 추가 - .insertBefore()

const para2 = document.createElement("p") => p 태그 생성
const node2 = document.createTextNode("[NEW!] 문단 3.") => 노드 생성
para2.appendChild(node2) => p 태그에 노드 추가

const elmt2 = document.getElementById("div2") => 원위치 찾기(div2)
const child2 = document.getElementById("p3") => 원하는 위치 찾기(p3)
elmt2.insertBefore(para2, child2) => 원하는 위치 전에 새요소 추가

자식 노드 제거- .removeChild()

const parent = document.getElementById("div4") => 부모 노드 찾기
const child = document.getElementById("p7"); => id="p7"인 자식 요소 찾기
parent.removeChild(child); => 부모 노드에서 자식 요소 제거

자식 노드 제거- .parentNode.removeChild()

* 부모 노드에서 자식 노드 찾아 제거

const child3 = document.getElementById("p7") => 자식 요소 찾기
child3.parentNode.removeChild(child3) => 자식 요소의 부모 노드 찾아 자식 제거
