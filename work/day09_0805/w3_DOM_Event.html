onload / onunload 이벤트

* onload/onunload: 사용자가 페이지에 들어오거나 나갈 때 트리거
* onload는 방문자의 브라우저 유형과 브라우저 버전을 확인하고, 해당 정보에 따라 적절한 버전의 웹 페이지를 로드하는 데 사용

oninput 이벤트

*oninput 이벤트: 사용자가 데이터를 입력하는 동안 어떤 동작이 수행되는 경우
* 아래 입력 필드를 작성할 때, 대문자로 변환 함수 트리거 

            input 태그 안에 => oninput="함수명()"

            function 함수명() {
                const x =
                    document.getElementById("fname")
                x.value = x.value.toUpperCase()
            }

            function 함수명() {
